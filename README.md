# mitdonga-next

A Next.js portfolio website built with TypeScript, React, and Tailwind CSS.

**Live Site**: [mitdonga.com](https://mitdonga.com)

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Technologies

- Next.js
- TypeScript
- React
- Tailwind CSS
- shadcn-ui
